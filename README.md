# 书店TPS
书店交易处理系统 (Transaction Processing System), 为帮助书店管理客户信息, 图书信息, 客户购买图书的交易订单信息。

## 业务流程建模
Business Process Modelling: 用户购买图书, 每条订单明确以下内容: 书籍名称和购买书籍的数量，例如：

```json
{购买者ID: customer_001, 购买图书ID: book_018, 购买数量: 2, 图书价格: $15.00} 
```
